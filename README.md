# Flipkart-Sales-Performance-Dashboard-2025
Interactive Power BI dashboard built using Flipkart sales data. Includes KPI cards, DAX measures, Power Query transformations, slicers, and visualizations to analyze sales, profit, orders, categories, cities, payment methods, and monthly trends.

**Project Overview**
Developed an interactive Power BI dashboard to analyze Flipkart sales performance for 2025. The dashboard provides insights into sales, profit, orders, category performance, city-wise sales, payment methods, and monthly trends.

**Features**
KPI Cards (Total Sales, Profit, Profit Margin, Orders)
Sales by Category (Treemap)
Quarterly Sales Analysis
Monthly Sales Trend
Top 5 Cities by Sales
Payment Method Distribution
Date Range Slicer
Interactive Filters

**Tools Used**
Power BI
Power Query
DAX
Excel
